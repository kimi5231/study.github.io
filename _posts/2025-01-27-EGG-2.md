{% capture notice-1 %}
#### revert

* 버전을 되돌린 새로운 버전을 만드는 것
* 커밋을 되돌릴 때 사용
{% endcapture %}

{% capture notice-2 %}
#### reset

* 버전을 완전히 되돌리는 것
* 지정한 커밋까지 현재 브랜치를 초기화할 때 사용
* soft reset, mixed reset, hard reset으로 세 종류
{% endcapture %}

{% capture notice-3 %}
* soft reset: 커밋만 되돌리는 것
* mixed reset: 스테이지까지 되돌리는 것
* hard reset: 작업 디렉터리까지 되돌리는 것
{% endcapture %}
